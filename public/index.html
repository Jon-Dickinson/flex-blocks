FLEX BLOCKS

Flexbox based custom reusable naming pattern

ver 1.0.0

Containing Blocks

The class name describes the layout mode and how the containing block should be used or how child elements will be positioned.

.if-base-wrapper { position: relative; display: inline-flex; width: 100%; } .flex-mid-container { position: relative; display: flex; margin-left: auto; margin-right: auto; padding-left: 20px; padding-right: 20px; width: 100%; } <div class="if-base-wrapper"> <div class="flex-mid-container max-w-75-perc"> </div> </div>
.horizontal-inline-flex { position: relative; display: inline-flex; width: 100%; } <div class="if-base-wrapper"> <div class="flex-mid-container max-w-75-perc"> <div class="horizontal-inline-flex"> </div> </div> </div>
.vertical-inline-flex { position: relative; display: inline-flex; flex-direction: column; width: 100%; } <div class="if-base-wrapper"> <div class="flex-mid-container max-w-75-perc"> <div class="horizontal-inline-flex"> <div class="vertical-inline-flex"> </div> <div class="vertical-inline-flex"> </div> </div> </div> </div>

Children of Containing Blocks

The class name describes the layout mode and it indicates that it should be used to contain content.

.if-content-panel { position: relative; display: inline-flex; } .flex-content-panel { position: relative; display: flex; } /* for images CSS2 */ .ib-content-panel { position: relative; display: inline-block; } <div class="if-base-wrapper"> <div class="flex-mid-container max-w-75-perc"> <div class="horizontal-inline-flex"> <div class="vertical-inline-flex"> <div class="if-content-panel"> </div> <div class="if-content-panel"> </div> </div> <div class="vertical-inline-flex"> <div class="if-content-panel"> </div> <div class="if-content-panel"> </div> </div> </div> </div> </div>

Modifier classes

Contains one property that is compatible with other properties and the class name describes the property and its value.

.direction--row { flex-direction: row !important; } .direction--column { flex-direction: column !important; } <div class="if-base-wrapper direction--column"> <div class="flex-mid-container direction--column max-w-75-perc"> <div class="horizontal-inline-flex"> <div class="vertical-inline-flex"> <div class="if-content-panel"> </div> <div class="if-content-panel"> </div> </div> <div class="vertical-inline-flex"> <div class="if-content-panel"> </div> <div class="if-content-panel"> </div> </div> </div> </div> </div>

Utility classes

A utility class contains one property and the class name describes the property and its value.
