FLEX BLOCKS

Flexbox based custom reusable naming pattern

ver 1.0.0

Containing Blocks

The class name describes the layout mode and how the containing block should be used or how child elements will be positioned.

.if-base-wrapper { position: relative; display: inline-flex; width: 100%; } .flex-mid-container { position: relative; display: flex; margin-left: auto; margin-right: auto; padding-left: 20px; padding-right: 20px; width: 100%; } .vertical-inline-flex { position: relative; display: inline-flex; flex-direction: column; width: 100%; } .horizontal-inline-flex { position: relative; display: inline-flex; width: 100%; }

Children of Containing Blocks

The class name describes the layout mode and it indicates that it should be used to contain content.

.if-content-panel { position: relative; display: inline-flex; } .flex-content-panel { position: relative; display: flex; } /* for images CSS2 */ .ib-content-panel { position: relative; display: inline-block; }

Modifier classes

Contains one property that is compatible with other properties and the class name describes the property and its value.

.direction--row { flex-direction: row !important; } .direction--column { flex-direction: column !important; }

Utility classes

A utility class contains one property and the class name describes the property and its value.

.margin-t-5 { margin-top: 5px !important; } .border-radius-4 { border-radius: 4px !important; } .border-1px-ffffff { border: 1px solid #ffffff !important; } .top-10 { top: 10px !important; }
